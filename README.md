## 3.管理队列
### ①加入心跳机制
```
成功登陆后，开启心跳检测
```
### ②掉线断开连接
```
掉线关闭程序，使用管道同学关闭信号
```
### ③断线重连机制
```
1.模拟建立通信
2.在通信过程中断开连接
3.期间不断建立通信
4.断开的client带着userId重连
5.恢复之前的通信
```
### ④断线后从连接队列中删除
